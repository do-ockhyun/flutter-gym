### UI 
- page : 최상의 화면
- layout : 재사용성 위젯 모음 혹은 복잡하여 별도 관리 위젯 모음
- widget : 단일 기능, 기본형과 커스텀 으로 분리

화면설계서를 출력하고, 위에서 아래로, 왼쪽에서 오른쪽으로 
- 상단에 page 
- 왼쪽에 Layout 
  - 주로 한 행으로 구성
- 오론쪽에 widget
